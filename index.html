Team memory for Claude Code

manymems captures intent, decisions, and code provenance across every session — so your team never re-discovers what it already knows.

Open Dashboard → Read the Docs

What it does

⚙

Persistent memory

Every session is captured and indexed. Ask what changed last week — get a real answer.

🔍

Semantic search

Find past decisions, bug fixes, and architectural choices with natural-language queries.

👥

Team provenance

Every code change linked to the intent behind it — who decided what, and why.

⚡

Zero-friction capture

Automatic observation capture via Claude Code hooks — no manual notes required.

🕐

Timeline view

Browse the full history of a file, feature, or decision with provenance drill-down.

🔗

Multi-platform

Works with Claude Code, Cursor, Gemini CLI, and any AI coding assistant via MCP.

Quick start

claude mcp add manymems npx @chicago-joe/manymems

Then open the dashboard at localhost:37778 or read the installation guide.
